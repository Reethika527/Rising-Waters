# Rising Waters

Rising Waters is a machine learning-based flood prediction system designed to identify possible flood risks using environmental and weather-related parameters. The project uses historical rainfall and climate data to generate predictions that help support disaster management and early warning systems.

## Features
- Flood Prediction using Machine Learning
- Data Preprocessing and Feature Scaling
- Model Comparison (Decision Tree, Random Forest, KNN, XGBoost)
- Flask Web Application
- Real-Time Prediction Interface

## Technologies Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Flask
- HTML/CSS
