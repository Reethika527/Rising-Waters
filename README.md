# Rising Waters 🌊

Rising Waters is a Machine Learning based Flood Prediction System developed to identify possible flood risks using rainfall and weather conditions.

## Features
- Flood Prediction using ML
- Flask Web Application
- Data Preprocessing
- Feature Scaling
- Model Comparison
- Real-time Prediction

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- XGBoost
- HTML
- CSS
- JavaScript

## Machine Learning Models
- Decision Tree
- Random Forest
- KNN
- XGBoost

## Project Structure

RISING-WATERS/
│
├── app.py
├── floods.save
├── transform.save
├── templates/
├── static/

## How to Run

1. Install Flask
2. Run:
python3 app.py

3. Open browser:
http://127.0.0.1:5001

## Output
The system predicts whether flood chances are present or not based on weather conditions.
